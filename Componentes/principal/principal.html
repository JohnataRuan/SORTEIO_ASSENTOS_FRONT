Realize Sorteio de Assentos com Facilidade

Este sistema permite que você realize o sorteio de assentos de forma fácil e prática.
Com ele, é possível sortear as salas de forma aleatória, gerar listas de presença para cada sala, criar um mapa visual com a posição de cada aluno e gerar uma lista detalhada com a localização de cada aluno em cada sala.

Importação de Arquivos

Para iniciar o sorteio de assentos, você pode importar uma planilha no formato CSV contendo os nomes dos alunos e suas respectivas turmas. O sistema usará essa lista para gerar o mapa de assentos e organizar os estudantes nas salas correspondentes.

Download de Arquivos

Baixe os arquivos do sorteio de turmas conforme as configurações selecionadas. Para realizar o sorteio, selecione ao menos duas salas. Observação: turmas do Ensino Médio e Ensino Fundamental não podem ser misturadas.
