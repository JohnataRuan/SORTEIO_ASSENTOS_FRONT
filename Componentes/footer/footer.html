O que é o programa ?

O programa é uma solução automatizada para o sorteio de assentos em salas de aula, fornecendo uma maneira fácil e prática de organizar a distribuição de alunos e gerar listas de presença para os momentos de prova.

Qual problema será solucionado?

Este software foi desenvolvido por alunos Tecinfo com o objetivo de resolver a complexa tarefa de sorteio de salas para provas na escola Lourdinas, abrangendo tanto o Ensino Fundamental quanto o Ensino Médio. A ferramenta visa proporcionar um método de acesso simples e eficiente para organizar a alocação dos alunos nas salas, garantindo uma distribuição justa e aleatória.
